QC Checklist

  • ? Make sure that the booking was created in a Mondee PCC, if not, then you must check if the airline allows another PCC other than Mondee's.
  • ? Confirm that types are written correctly, especially for LHG tickets. For example, it should be *CHD*23JUL19 for CHD type on Sabre.
  • ? Check the dates and the codeshare flights and which airline should be the validating carrier (over the world).
  • ? Check if the agent has left any info for the fare needed in the PNR.
  • ? Check if there's a difference between *COM and normal pricing. Also, verify the accuracy of the pricing and whether it's what the agent has asked for.
  • ? Make sure you know what's the right plus-up/COMM/TC/TD because you'll need it later when QCing the ticket.
  • ? Confirm that CTCE/CTCM and DOCs exist, especially for airlines that require so like SV and TK.
  • ? Make sure that DK/Customer Number exists on the booking.
  • ? Make sure that ticket's coupons reflect the booking's segments.
  • ? Confirm the number of bag pieces on each coupon and whether it meets the agent's request.
  • ? Confirm the ticket has the correct base fare and total value (in one case, a plus-up was added but it just didn't reflect as COMM on Amadeus).
  • ? Confirm that the correct plus-up/COMM/TC/TD is reflecting on the ticket.
  • ? Confirm that the correct FOP is used on the ticket as well. Check if it's CASH or with CC, and if it's CASH, is it allowed? (for the airline or the agent).
